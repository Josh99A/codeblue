## Current Direction

The current logo system uses:

- a `CB` monogram with an ECG pulse to signal responsiveness and continuity of care
- deep navy as the primary trust color
- teal as the clinical-tech accent
- horizontal and symbol-only variants suited for website headers, app icons, and brand review
